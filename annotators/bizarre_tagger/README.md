# bizarre_tagger

**Conda env:** `live2d_ann_detectron2`

## Setup

```bash
conda create -n live2d_ann_detectron2 python=3.10
conda activate live2d_ann_detectron2
pip install git+https://github.com/facebookresearch/detectron2@main#egg=detectron2 \
  --no-build-isolation
```
